# InclusivFood 🌾

## 📋 Sobre o Projeto
O **InclusivFood** é uma aplicação web focada em segurança alimentar adaptada para pessoas em vulnerabilidade nutricional ou restrições severas. O sistema foi desenvolvido como projeto para o Concurso de Análise e Desenvolvimento de Sistemas (2026).

## 🛠️ Tecnologias Utilizadas
* HTML5 estrutural customizado
* CSS3 responsivo com layout Grid e Flexbox
* JavaScript Moderno Nativo (Vanilla JS)
* Integração em tempo real com a **API v2 da Open Food Facts**
